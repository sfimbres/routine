A minimal web app that shows a rotating motivational message and reveals today's workout routine.

Core functionality
- Dynamically display a random motivational message inside the #message container using the [`generateMessage`](js/main.js) function ([js/main.js](js/main.js)).
- Show only today's workout section by revealing the matching day div (e.g. `#monday`, `#tuesday`, ...) using the [`highlightTodayWorkout`](js/main.js) function ([js/main.js](js/main.js)).
